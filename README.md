# ML Мини-сервис: Классификация тональности + Мониторинг дрифта

Простой ML-сервис, который классифицирует тональность отзывов с помощью FastAPI, логирует предсказания в SQLite, симулирует продакшен-трафик с дрейфом данных и отслеживает ухудшение качества со временем.

## Что внутри

| Компонент | Описание |
|-----------|----------|
| `src/generate_data.py` | Генерация синтетических данных: train/test + 10 "дней" продакшена с имитацией дрифта |
| `src/train.py` | Обучение модели (TF-IDF + LogisticRegression), сохранение метрик |
| `src/app.py` | FastAPI-сервис: эндпоинты `/health`, `/predict`, `/predict/history`, логирование в SQLite |
| `src/simulate_traffic.py` | Симуляция продакшен-трафика: отправка 200 запросов в сервис |
| `src/monitor.py` | Мониторинг качества: метрики по дням, детекция дрифта, алерты, график |

## Быстрый старт

```bash
# 1. Установка зависимостей
pip install -r requirements.txt

# 2. Генерация данных
cd src && python generate_data.py

# 3. Обучение модели
python train.py

# 4. Запуск сервиса (в отдельном терминале)
uvicorn app:app --reload

# 5. Симуляция трафика (в другом терминале)
python simulate_traffic.py

# 6. Мониторинг качества
python monitor.py
```

## Эндпоинты API

- `GET /health` — проверка живости
- `POST /predict` — предсказание тональности (`{"text": "..."}`)
- `GET /predict/history?limit=20` — история предсказаний

## Мониторинг

Скрипт `monitor.py`:
- Джойнит логи предсказаний с "истинными" метками (имитация задержанной разметки)
- Считает accuracy, F1, avg_confidence по дням
- Детектирует деградацию по порогам
- Сохраняет отчёт в `reports/monitoring_report.json`
- Строит график в `reports/quality_over_time.png`
